# pfSense開源防火牆
```
pfSense是一個基於FreeBSD的開源防火牆及路由平台（即作業系統）。

pfSense可安裝於物理電腦或虛擬機器，能夠在網路中充當獨立的防火牆及路由器。
其配備用於進行設定及更新等管理工作的Web用戶介面，因而易用性較高。

pfSense專案作爲m0n0wall專案的分支起始於2004年，並在2006年釋出了首個版本。
其名稱衍生自資料包過濾工具PF。

該項目於2018年9月24日發布了2.4.4版。

https://zh.wikipedia.org/wiki/PfSense

https://www.netgate.com/training/
```
